# 快捷键

## 比较熟悉的快捷键

```python
--- 高频
Ctrl + W				# 关闭当前窗口
Ctrl + D				# 添加书签
Ctrl + H				# 打开历史记录
Ctrl + P				# 打印当前页面(有时候选中打印效果更好)
Ctrl + R				# 重新加载
Ctrl + N				# 新建窗口

--- 低频
Ctrl + S				# 网页另存为
```

## 需要记住的快捷键

```python
Ctrl + T				# 新建标签页
Ctrl + J				# 查看下载内容
Ctrl + Shift + T		# 查看下载内容
Ctrl + Shift + N		# 新建无痕窗口
Ctrl + Shift + O		# 书签管理器
```

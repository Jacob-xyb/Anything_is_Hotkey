# 通用快捷键
```python
--- 高频
Space					# 向下翻页
Shift + Space			# 向上翻页
Ctrl + W				# 关闭当前窗口
Ctrl + D				# 添加书签
Ctrl + H				# 打开历史记录
Ctrl + P				# 打印当前页面(有时候选中打印效果更好)
Ctrl + R				# 重新加载
Ctrl + N				# 新建窗口
Ctrl + T				# 新建新标签页
Ctrl + L				# 光标定位到地址栏
Ctrl + +				# 页面放大
Ctrl + -				# 页面缩小
Ctrl + 0				# 页面恢复默认
Ctrl + Shift + N		# 新建无痕窗口（Chrome）
Ctrl + Shift + P		# 新建无痕窗口（QQ）

--- 低频
Ctrl + S				# 网页另存为
Ctrl + J				# 查看下载内容
Ctrl + Shift + T		# 查看下载内容（Chrome）
Ctrl + Shift + T		# 恢复最近关闭的标签（QQ）
Ctrl + Shift + O		# 书签管理器
```

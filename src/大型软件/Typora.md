# 快捷键

## Windows


- 比较熟悉的快捷键

```python
--- 高频
Ctrl + /										# 切换源代码格式  
Ctrl + W										# 关闭当前窗口
Ctrl + C										# 复制当前行
Ctrl + X										# 剪切当前行
Ctrl + B										# 字体加粗(选中时会对选中的加粗，是加**的形式)
Ctrl + Left/Right						# 字句左/右平移
Home/End										# 当前行首/尾
Ctrl + Home/End							# 当前文本首/尾


# 文件树视图
Ctrl + 单击				# 以新窗口打开文件

--- 低频
Ctrl + T				# 创建表格
Ctrl + L				# 选中当前行
Ctrl + D				# 选中当前词汇
Ctrl + E				# 选中当前格式文本
Ctrl + 1~6				# 1~6级标题
Ctrl + -/+				# 降低/提升标题等级
```

## macOS

- 比较熟悉的快捷键

```python
---高频
command + /									# 切换源代码格式
command + w									# 关闭当前窗口
command + c									# 复制当前行
command + x									# 剪切当前行
command + b									# 字体加粗(选中时会对选中的加粗，是加**的形式)
command + left/right				# 当前行首/尾
option + left/right					# 字句左/右平移
command + up/down						# 当前文本首/尾

# 文件树视图
command + 单击								# 以新窗口打开文件

---低频
command + l									# 选中当前行
command + d									# 选中当前词汇
command + e									# 选中当前格式文本（macOS有bug：在代码文本中不仅不会选中全文，同时还会锁键盘使键盘失效。）
command + 1~6								# 1~6级标题
command + -/+								# 降低/提升标题等级
```

- 不常用

```python
command + shift + d					# 删除当前词
```

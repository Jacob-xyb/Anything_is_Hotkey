# 快捷键
```python
Alt + Enter				# 选择忽略PEP8代码风格错误

alt + j					# 将下一个匹配项添加到选择  # *2	

ctrl + enter			# 拆分行

Ctrl + D				# 快速复制当前行到下一行

Shift + F6				# pycharm变量统一命名

Shift + Enter			# 开始新行

Shift + Shift			# 全局搜索

Ctrl + Shift + Z		# 前进

Ctrl + Shift + R		# 全局变量替换

Ctrl + Shift + F		# 当前项目中查找
ctrl + shift + up/down		# 向上/下移动语句
Ctrl + Alt + Left/Right		# 追踪光标位置（与网易云全局搜索快捷键冲突） # *1
```
